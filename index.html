HTML Basics

HTML (HyperText Markup Language) is the standard markup language for creating web pages.

  • Provides the basic structure of a webpage.
  • Supports multimedia elements like images, videos, and audio.

HTML Elements

HTML elements are the building blocks of HTML documents.

  • Elements consist of an opening tag, content, and a closing tag.
  • They can be nested and styled using CSS.

HTML Attributes

Attributes provide additional information about an element.

  • Attributes are specified in the opening tag.
  • Common attributes include id, class, src, and alt.
